Responsive Designtester

Features:

  • Navigation Bar
  • Custom Resolutions
  • Resolutions Presets
  • 'Fullscreen'
  • OpenSource

Usage:

This tool is meant to be used by Web Designers and Developers to test their online or localhost web applications.

Type website in address bar to navigate. Use size input fields to define custom frame size or use presets button besite to choose from popular resolutions.

Use mouse to drag edges or corner of the frame.

Hold SHIFT to enable snap to 10 pixels and slow down animation on frame size changes.
